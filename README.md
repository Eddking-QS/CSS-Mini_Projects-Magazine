# Mini Project: Magazine

## Description

A CSS project to practice CSS Grid. This project is a simple webpage with a magazine layout. The project is designed to be responsive and uses CSS Grid to create the layout.

## Features

-   CSS Grid
-   Responsive Design
-   Media Queries
-   Pseudo-elements
